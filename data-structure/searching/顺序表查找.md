# sequence search
```c++
int sequence_searching(vector<int> seqList, int key) {
  for (int i = 0; i < seqList.size(); i++) {
    if (seqList.at(i) == key) return i;
  }
  return -1;
}
```

优化
```c++
int squence_searching(vector<int> seqList, int key) {
  int i = seqList.size() - 1;
  while (seqList.at(i) != key) i--;
  return i;
}
```
